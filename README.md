# OhmMaster Pro v2.0 🚀

**OhmMaster Pro** es una suite de ingeniería móvil diseñada para la resolución rápida de circuitos electrónicos. Desarrollada bajo el estándar *Premium Engineering*, combina una estética visual avanzada con alta precisión matemática para uso profesional en campo.

## 📱 Funcionalidades Principales

- **Calculadora de Ley de Ohm:** Resolución automática de Voltaje (V), Corriente (I), Resistencia (R) y Potencia (P) partiendo de cualquier par de valores.
- **Divisor de Voltaje:** Cálculo de Vout basado en Vin, R1 y R2.
- **Optimización de Prefijos:** Teclado inteligente integrado que permite el ingreso fluido de unidades de ingeniería (G, M, k, 1, m, µ, n, p).
- **Diseño Glassmorphism:** Interfaz oscura optimizada para visibilidad y bajo consumo de batería, con soporte nativo para Safe Areas.

## 🛠️ Stack Tecnológico

- **Core:** [Vite](https://vitejs.dev/) (Vanilla JavaScript)
- **Bridge Nativo:** [Capacitor](https://capacitorjs.com/)
- **Estilos:** CSS3 nativo con Mobile-First Design y Backdrop Filters.
- **CI/CD:** GitHub Actions para compilación automatizada de APKs.

## 🚀 Instalación y Desarrollo

### Requisitos
- Node.js 18+
- Android Studio (para emulación local)

### Comandos Rápidos
```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Compilar assets web
npm run build

# Sincronizar con Android
npx cap sync android
```
